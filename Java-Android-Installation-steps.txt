-------------------------------------------- JDK --------------------------------------------
* For 64 bit:
https://www.oracle.com/technetwork/java/javase/downloads/jdk13-downloads-5672538.html

* For 32 bit:
https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
In the above link select "Windows x86". It make take you to 'Oracle login' page.

After installing JAVA, run below commands in CMD to check whether JAVA is installed properly or not.
* java
* java --version
* javac

* After installing JDK, just add below path inside of 'path' variable of "System variables"
	-> C:\Program Files\Java\jdk-13.0.1\bin
* Then run below commands in CMD to confirm java installation
	-> java
	-> javac
	-> java --version
